About This Project

This web application was developed to visualize and analyze COVID-19 data from around the world. Using D3.js, HTML, CSS, and JavaScript, it transforms raw data from a CSV file into interactive visualizations that help users understand the global impact of the pandemic.

The visualizations explore relationships between various factors such as healthcare resources, demographics, geographical location, and COVID-19 outcomes.

By researching and applying the basic building blocks used to create web applications, I was able to go through the learning curve, and now have the ability to create more complex applications.
